# UI কপি — উৎস সত্য (single source)

এই ফাইলে শুধু **দুইটি কলাম** আছে:

| প্রসঙ্গ | চূড়ান্ত কপি |
|---------|---------------|

- **প্রসঙ্গ**: কোথায় / কী জিনিস (কোডে খুঁজতে)।  
- **চূড়ান্ত কপি**: অ্যাপে যা দেখাবে / পাঠাবে — **এটাই** পরবর্তীতে কোডে বসানোর মানদণ্ড।  
- খালি রাখতে চাইলে **`খালি`** বা **`-`** লিখুন (কোডে শূন্য স্ট্রিং বা লুকানো — আলাদা বলে দিলে হবে)।  
- যখন ইচ্ছে এই ফাইল এডিট করুন; আপডেট চাইলে এজেন্টকে বললেই এখান থেকে কোড সিঙ্ক করা যাবে।

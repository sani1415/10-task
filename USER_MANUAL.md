# Waqful Madinah — ব্যবহারকারী ম্যানুয়াল

> এটি **শিক্ষক ও ছাত্র**-দের জন্য। কোড/ডেভেলপমেন্ট নিয়ম `CLAUDE.md`-তে আছে — সেটা আলাদা।

---

## ১. অ্যাপ কীভাবে চালু করবেন

### সাধারণ ব্যবহার (ইন্টারনেটে ডিপ্লয় করা অ্যাপ)

1. ব্রাউজারে অ্যাপের ঠিকানা খুলুন (যেমন Vercel-এ হোস্ট করা URL)।
2. প্রথম পেজে **শিক্ষক** বা **ওয়াকিফ (ছাত্র)** বেছে নিন।
3. পরের বার একই ডিভাইসে সরাসরি শেষ ব্যবহার করা প্যানেলে যেতে পারে।

**ফোনে অ্যাপের মতো ব্যবহার:** ব্রাউজারের **ইন্সটল** / **হোম স্ক্রিনে যোগ করুন** অপশন দিন। তাহলে আইকন থেকে খোলা যায়।

**নোটিফিকেশন:** লক স্ক্রিনে বেল আইকন বা লগইনের পর অনুমতি দিন — নতুন রিসালা/বার্তার জন্য।

---

### কম্পিউটারে টেস্ট / প্রিভিউ (শিক্ষক বা রক্ষণাবেক্ষণকারী)

প্রজেক্ট ফোল্ডারে টার্মিনাল খুলে:

```powershell
npm install
npm start
```

তারপর ব্রাউজারে:

- ছাত্র: `http://localhost:3000/student.html`
- শিক্ষক: `http://localhost:3000/teacher.html`

**মনে রাখুন:** Cursor-এর Live Server (Go Live) দিয়ে পুরো অ্যাপ ঠিকমতো চলে না। টেস্টের জন্য **`npm start`** ব্যবহার করুন।

**ইন্টারনেট ছাড়া:** শুধু এক ব্রাউজারে ডেমো — ডেটা সবার মাঝে শেয়ার হবে না।  
**ইন্টারনেটসহ (Supabase):** সব ডিভাইসে একই ডেটা — শিক্ষক ও ছাত্র আলাদা ডিভাইসে লগইন করলে সিঙ্ক হয়।

## ২. লগইন

| প্যানেল | কী লাগে |
|--------|---------|
| **শিক্ষক** | ৪ অঙ্কের শিক্ষক পিন |
| **ছাত্র** | ওয়াকফ নং (যেমন `001`) + ৪ অঙ্কের পিন |

- একই পিন একাধিক ছাত্রে থাকতে পারে — লগইন **ওয়াকফ নং + পিন** জোড়ায় হয়।
- ভুল পিন/আইডি দিলে লগইন হবে না।
- **লগআউট:** উপরে ডান পাশের বাটন।

---

## ৩. ছাত্র প্যানেল (ওয়াকিফ)

লগইনের পর প্রথমে **হোম** খোলে। নিচে ৬টি মেনু।

### হোম
- **অ্যালার্ট:** নতুন রিসালা, বাকি পরীক্ষা, বাকি আমল — ট্যাপ করলে সেই জায়গায় যাবে।
- **আজকের আমল:** আজ কতটা সম্পন্ন — **বিস্তারিত** থেকে আমল ট্যাবে যান।
- **দৈনিক সময়সূচি:** দিনের রুটিনের সংক্ষিপ্ত তালিকা — **সম্পূর্ণ** থেকে সম্পাদনা/দেখা যায়।

### রিসালা
- উস্তাদকে লিখিত বার্তা পাঠানো।
- ফাইল সংযুক্তি (ছবি, PDF ইত্যাদি) — সর্বোচ্চ **১০ MB**।
- উস্তাদের পাঠানো আমল/ফাইল এখানেও দেখা যায়।
- উস্তাদ পড়লে টিক দেখা যায়।
- **খোঁজ:** উপরে 🔍 — রিসালা খুঁজুন।

### আমল
তিনটি উপ-ট্যাব:

| ট্যাব | কী করা যায় |
|-------|------------|
| **আজ** | দৈনিক ও এককালীন আমল দেখা, আজ সম্পন্ন মার্ক করা |
| **ইতিহাস** | ক্যালেন্ডারে গত দিনের আমল |
| **সময়সূচি** | পুরো দৈনিক সময়সূচি দেখা ও **সম্পাদনা** — পরিবর্তন উস্তাদ অনুমোদন দিলে কার্যকর |

**ছাত্র পারে:** আমল সম্পন্ন মার্ক, সময়সূচি প্রস্তাব পাঠানো।  
**ছাত্র পারে না:** নিজে আমল তৈরি বা মুছে ফেলা।

### পরীক্ষা
- উস্তাদের দেওয়া পরীক্ষা দেখা ও **জমা দেওয়া**।
- সময়সীমা থাকলে টাইমার চলে।
- একবার জমা দিলে আবার দেওয়া যায় না (উস্তাদ নতুন পরীক্ষা দিলে আলাদা)।

### লক্ষ্য
- নিজের লক্ষ্য যোগ, সম্পন্ন মার্ক, মুছে ফেলা।
- বিভাগ: ইলম, আমল, আখলাক, আফকার।

### ফাইল
- **প্রেরিত:** উস্তাদের কাছে ফাইল পাঠানো (রোযনামচা, বিবরণ ইত্যাদি)। একাধিক ছবি একটি PDF-এ জোড়া যায়।
- **প্রাপ্ত:** উস্তাদের পাঠানো ফাইল দেখা।

### প্রোফাইল (উপরে বাম অবতার)
- নাম/ওয়াকফ আইডি দেখা।
- **পিন পরিবর্তন** (৪ অঙ্ক) — পুরনো পিন যাচাই হয়।

## ৪. শিক্ষক প্যানেল

শিক্ষক **একটি ডিভাইস** ব্যবহার করেন; ছাত্ররা **আলাদা ডিভাইস**। নিচে ৬টি মেনু।

### চ্যাট
- ছাত্র তালিকা — কারো চ্যাট খুলুন।
- **গ্রুপ বার্তা:** সব ছাত্রকে একসাথে (ব্রডকাস্ট)।
- **গ্রুপ:** নির্দিষ্ট ছাত্রদের ট্যাগ/গ্রুপে বার্তা।
- চ্যাটের ভিতরে ছাত্রের **আমল, ডকুমেন্ট, পরীক্ষা, লক্ষ্য, নোট** ট্যাব দিয়ে দেখা।
- ছাত্র **সময়সূচি পরিবর্তন** প্রস্তাব দিলে চ্যাটে অনুমোদন/প্রত্যাখ্যান।
- ফাইল পাঠানো, রিপ্লাই, খোঁজ।

**উপরে ডায়েরি (📓):** শিক্ষকের ব্যক্তিগত ডায়েরি — **ছাত্র দেখতে পারে না**।

### ছাত্র
- নতুন ছাত্র যোগ (**＋** বাটন)।
- তালিকা থেকে **প্রোফাইল** খুলুন:

| প্রোফাইল ট্যাব | কী করা যায় |
|----------------|------------|
| **তথ্য** | নাম, ওয়াকফ আইডি, পিন, রঙ, দায়িত্ব ইত্যাদি |
| **সময়সূচি** | ছাত্রের দৈনিক সময়সূচি সেট/সম্পাদনা |
| **একাডেমিক** | পড়ার ইতিহাস |
| **নোট** | শিক্ষকের গোপন নোট (ছাত্র দেখে না) |
| **বিকল্প** | ডেটা মুছা, ছাত্র সম্পূর্ণ মুছে ফেলা ইত্যাদি |

### আমল
- **তালিকা:** আমল তৈরি, ছাত্র বরাদ্দ, দৈনিক/এককালীন, ফিল্টার।
- **সারসংক্ষেপ:** সব ছাত্রের আমলের চিত্র।
- **ছাত্রভিত্তিক:** কে কতটা করেছে।

**＋** বাটন দিয়ে নতুন আমল যোগ।

### পরীক্ষা
- **সব পরীক্ষা:** তালিকা, সম্পাদনা, মুছে ফেলা।
- **নতুন:** প্রশ্ন তৈরি, ছাত্র বেছে নেওয়া, সময় ও পাস নম্বর।
- **ফলাফল:** কে কত পেয়েছে দেখা।

### ডকুমেন্ট
- **ছাত্রের পাঠানো:** পর্যালোচনা, ডাউনলোড।
- **আমার পাঠানো:** ছাত্রকে ফাইল পাঠানোর ইতিহাস।
- **স্টোরেজ:** ফাইল স্টোরেজ দেখা।

### সেটিং
- শিক্ষকের নাম, মাদরাসার নাম।
- শিক্ষক **পিন পরিবর্তন**।
- **ব্যাকআপ / পুনরুদ্ধার** (JSON) — গুরুত্বপূর্ণ ডেটা সংরক্ষণ।
- ছাত্র প্যানেলে যাওয়ার লিংক।

## ৬. সিঙ্ক ও নোটিফিকেশন

- **অনলাইন + Supabase:** অ্যাপ খোলা থাকলে ডেটা আপডেট হয় (রিসালা, আমল, পরীক্ষা ইত্যাদি)।
- **অ্যাপ বন্ধ থাকলে:** নোটিফিকেশন অনুমতি দিলে নতুন বার্তার পুশ আসতে পারে (ডিভাইস সাপোর্ট করলে)।
- **একই ছাত্র দুই ফোনে:** একই ওয়াকফ আইডি + পিনে লগইন করলে একই ডেটা দেখবে।
